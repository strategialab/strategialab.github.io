Enviar o script

  1. No seu painel da Loja Integrada, acesse a área Upload de Arquivos;
  2. Clique no botão "Enviar um arquivo";
  3. Clique no botão "Escolher arquivo" e selecione o script que você fez o download;
  4. Clique no botão "Enviar arquivo" e aguarde alguns segundos até que a Loja Integrada processe o arquivo;
  5. Se tudo ocorrer bem, você deve ver a mensagem "Arquivo adicionado com sucesso."

Incluir Código HTML
