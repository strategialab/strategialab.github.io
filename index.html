Instalação

  1. Upload do Script
    1. No seu painel da Loja Integrada, acesse a área Upload de Arquivos;
    2. Clique no botão "Escolher arquivo" e selecione o script que você fez o download;
    3. Clique no botão "Enviar arquivo" e aguarde alguns segundos até que a Loja Integrada processe o arquivo;
    4. Se tudo ocorrer bem, você deve ver a mensagem "Arquivo adicionado com sucesso."
  2. Incluir Código HTML
    1. No seu painel da Loja Integrada, acesse a área Incluir Código HTML;
    2. Caso seja solicitado, confirme que leu o aviso e concordou;
    3. Preencha os dados do formulário com os seguintes dados:
      • Descrição: Barra de Frete Grátis
      • Local publicação: Rodapé
      • Página publicação: Página do carrinho
      • Tipo: Javascript
      • Conteúdo: clique aqui para copiar o código
    4. Clique no botão "Salvar código HTML"

Atualização

  1. No seu painel da Loja Integrada, acesse a área Incluir Código HTML;
  2. Copie e cole o novo código no campo "Conteúdo" do formulário: clique aqui para copiar o código
  3. Clique no botão "Salvar código HTML"
